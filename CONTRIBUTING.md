# Contributing to AetherIndex

We welcome contributions from the community to help make AetherIndex the sovereign standard for Solana indexing!

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Create a new branch** for your feature or bugfix.
3.  **Implement your changes** following the project's coding standards.
4.  **Run the verification suite** (`npx ts-node src/tests/production_proof.ts`) to ensure everything is solid.
5.  **Submit a Pull Request** with a clear description of your changes.

## Coding Standards

- Use TypeScript for all core logic.
- Prioritize performance and memory efficiency.
- Document all new features with appropriate comments and README updates.

## PRO Features

Note that the `pro/` directory is for internal premium features and is not part of the public infrastructure. Please do not submit pull requests targeting that directory.
